# UIKit-Estudo-de-Caso
Projeto de estudo focado em desenvolvimento iOS utilizando UIKit.
Este repositório foi criado com o objetivo de explorar e comparar diferentes formas de construção de interfaces em aplicações iOS, incluindo:

- UI Programática

- Storyboards

- Integração com SwiftUI utilizando UIHostingController

O projeto também possui foco em acessibilidade no desenvolvimento iOS.
Entre os recursos utilizados estão:

- Dynamic Type

- VoiceOver
